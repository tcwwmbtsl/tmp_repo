面向对象：Object Oriented Programming 
# 面向过程
    * 一种以过程为中心的思想
    * 首先分析解决问题所需要的步骤
    * 然后用函数将这些步骤一步一步的实现
    * 最后按顺序依次调用运行
# 面向对象
    * 是一种更符合我们人类思维习惯的编程思想
    * 面向对象开发就是不断的创建对象，使用对象，操作对象做事情
    * 可以将复杂的事情简单化
# 类对象
    * 类(class): 用来描述具有相同的属性和方法对象的集合，它定义了集合中每个对象所共有的属性和方法
    * 对象(object): 也称为类的实例，是一个具体存在的实体
